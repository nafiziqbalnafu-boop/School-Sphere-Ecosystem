# 🎓 School Sphere Ecosystem

🚀 A Smart Digital School Platform with AI Chat, Online Classes & Result System  
_Built by Nafiz Iqbal Pramanik & Team_

---

## 📌 Overview
School Sphere Ecosystem is a modern digital school management system that integrates AI-powered learning, online class management, and result tracking for both students and parents.

School Sphere হলো একটি complete smart platform যেখানে student, teacher এবং parent সবাই একটি ecosystem-এর মধ্যে যুক্ত থাকে।

---

## ✨ Core Features

### 🤖 AI Chatbot for Students
- Smart AI assistant for study help
- Ask questions and get instant answers
- Homework guidance & concept explanation

👉 AI helps students learn faster and smarter

---

### 📡 Online School (Live Class System)
- Live class via meeting links (Google Meet / Zoom)
- Class schedule system
- Future support for recorded classes

👉 Students can attend classes from anywhere

---

### 📊 Result Integration System
- Students can check results anytime
- Parents can also view student performance 👨‍👩‍👧
- Secure and real-time data system

👉 Transparent and easy academic tracking

---

## 👨‍🎓 Student Panel
- View results
- Check attendance
- Join live classes
- Chat with AI assistant

---

## 👨‍🏫 Teacher Panel
- Upload marks
- Manage classes
- Share notices
- Interact with students

---

## 🧑‍💼 Admin Panel
- Full system control
- Manage users (Student/Teacher/Parent)
- Monitor system activity

---

## ⚙️ Technologies Used
- HTML, CSS, JavaScript
- Firebase (Auth, Firestore, Hosting)
- AI API Integration

## 💡 Problem
Traditional school systems are manual, slow, and lack proper communication between students, teachers, and parents.

অনেক ক্ষেত্রে অভিভাবকরা (Parents) তাদের সন্তানের রেজাল্ট বা অগ্রগতি সহজে জানতে পারেন না।

---

## ✅ Solution
School Sphere provides a centralized digital platform where:
- Students learn with AI support
- Teachers manage classes online
- Parents track student progress easily

👉 One system, complete solution
